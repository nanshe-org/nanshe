nanshe.registerer module¶

The registerer module allows the registration algorithm to be run.

Overview¶

The main function actually starts the algorithm and can be called externally. Configuration files for the registerer are provided in the examples and are entitled registerer. Any attributes on the raw dataset are copied to the registered dataset.

API¶

nanshe.registerer.main(*args, **kwargs)[source]¶

Simple main function (like in C). Takes all arguments (as from sys.argv) and returns an exit status.

Parameters:argv (list) – arguments (includes command line call).
Returns:exit code (0 if success)
